# Meeting Follow-up Workflow

## Purpose

Turn conversations into clear decisions, actions, and accountability.

## Inputs

- Meeting notes
- Transcripts
- Voice notes
- Discussion points
- Open questions

## Workflow

Meeting Conversation

↓

AI-assisted Processing

- Summarize key discussion points
- Extract decisions made
- Identify action items
- Capture open questions
- Highlight risks or dependencies

↓

Human Review

- Validate important context
- Confirm priorities
- Adjust ownership
- Make final decisions

↓

Execution System

- Assign owners
- Set deadlines
- Create follow-up tasks
- Update project documentation

↓

Review Loop

- Track progress
- Close completed actions
- Capture lessons learned

## AI Role

AI reduces the effort required to capture and organize information, allowing more attention to be spent on judgment, prioritization, and decision-making.

## Output

- Meeting summary
- Decision log
- Action tracker
- Follow-up plan
- Updated project context
