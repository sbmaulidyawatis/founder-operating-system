# Inbox Triage Workflow

## Purpose

Reduce inbox overload by turning incoming information into clear priorities, actions, and follow-ups.

## Inputs

- Emails
- Messages
- Meeting invitations
- Requests from stakeholders
- Documents and references

## Workflow

Incoming Information

↓

AI-assisted Processing

- Summarize key context
- Identify action items
- Categorize urgency and importance
- Draft potential responses
- Highlight missing information

↓

Human Review

- Validate context
- Make decisions
- Adjust priorities
- Approve communication

↓

Execution System

- Create tasks
- Schedule follow-ups
- Update project trackers
- Document decisions

↓

Knowledge Base

- Store reusable information
- Capture decisions
- Improve future workflows

## AI Role

AI helps accelerate information processing, but final prioritization and decisions remain human-led.

## Output

- Clear action list
- Prioritized tasks
- Draft responses
- Follow-up reminders
- Documented decisions
